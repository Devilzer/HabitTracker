# HabitTracker
Habit tracker made using nodejs and express.
## [`Habit Tracker Demo`](https://habittracker-node.herokuapp.com/)
## Features
<ul>
  <li>User Signup/Signin.</li>
  <li>Track each habit everyday.</li>
  <li>Habits detail week view.</li>
  <li>Habit Streak.</li>
</ul>
